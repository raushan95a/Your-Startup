## Brand Colors

| Swatch | Hex | RGB | Usage |
|--------|-----|-----|-------|
| Off-white | `#FBFBFB` | rgb(251, 251, 251) | Page background |
| Light blue | `#E8F9FF` | rgb(232, 249, 255) | Section backgrounds, cards |
| Medium blue | `#C4D9FF` | rgb(196, 217, 255) | Highlights, badges, borders |
| Soft purple | `#C5BAFF` | rgb(197, 186, 255) | Accent color, CTAs, active states |

**Color usage guidelines:**
- Use `#FBFBFB` as the main page background — it's softer than pure white and reduces eye strain
- Use `#E8F9FF` for alternating section backgrounds (e.g. Services, Testimonials)
- Use `#C4D9FF` for card borders, tag chips, and subtle highlights
- Use `#C5BAFF` for primary buttons, active nav links, and accent elements
- Use dark text (`#1A1A2E` or `#2C2C4A`) on all light backgrounds for readability

## Design Notes for Developers

### Color Application

```css
:root {
  --color-bg-page:     #FBFBFB;  /* Main page background */
  --color-bg-section:  #E8F9FF;  /* Alternate section backgrounds */
  --color-accent-soft: #C4D9FF;  /* Card borders, tags, chips */
  --color-accent-main: #C5BAFF;  /* Primary buttons, active states */
  --color-text-dark:   #1A1A2E;  /* Main body text */
  --color-text-muted:  #5A5A7A;  /* Secondary text, captions */
}
```

### Typography Recommendations

- **Headings:** Inter or Plus Jakarta Sans (Google Fonts — free)
- **Body:** Inter Regular 400
- **Accent/tags:** Inter Medium 500
- **Base font size:** 16px body, 14px captions, 13px tags

### Button Styles
